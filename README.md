## Semi Autonomous Rescue Rover 

## Overview

The AI Semi Autonomous Rescue Rover is designed to assist rescue teams in disaster environments such as earthquakes, collapsed buildings, and hazardous zones.

The system integrates robotics, computer vision, environmental monitoring, and health sensing to detect survivors and hazards.

The rover uses an ESP32 microcontroller for navigation and an ESP32-CAM module for AI based human detection.

## Key Features

• AI based human detection using ESP32-CAM  
• Infrared temperature sensing for human detection  
• Heart rate & SpO2 monitoring  
• Obstacle avoidance  
• Pit detection  
• Gas and flame hazard detection  
• Sound based survivor detection  
• GPS location tracking  
• Semi autonomous navigation  
• Web based monitoring

---

## Applications

• Disaster rescue operations  
• Military surveillance  
• Hazardous environment monitoring  
• Industrial safety inspection
